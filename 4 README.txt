איך לפרוס את המערכת ב-Render.com (גם מהטלפון):

1. פתח דפדפן בטלפון וגש ל-https://render.com
2. צור חשבון חינמי (Google/GitHub).
3. ב-Dashboard לחץ על "New Web Service".
4. בחר "Manual Deploy" → "ZIP Upload".
5. גרור/העלה את ה-ZIP הזה (עם כל הקבצים שמופיעים למעלה).
6. Render יתקין תלותים ויפעיל את server.js.
7. תוך כמה רגעים תקבל קישור בצורת 
       https://your-service-name.onrender.com
8. פתח את הקישור, תתחבר – וכל המערכת פועלת בענן:
   - יצירת שלוחות
   - שכפול
   - איפוס + אימות כפול
   - מצב חי
   - לוג מלא
9. כדי להוסיף קיצור דרך אנדרואיד – לחץ על "⋮" (דפדפן Chrome) → "הוסף למסך הבית".

שיהיה בהצלחה!
